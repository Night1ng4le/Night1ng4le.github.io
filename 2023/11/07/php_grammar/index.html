PHP grammar

发表于 2023-11-07 分类于 Memo

弱类型语言，自动识别合适的类型。

  • 变量：

  • 输出

    • echo - 可以输出一个或多个字符串
    • print - 只允许输出一个字符串，返回值总为 1
    • 注：echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。

  • EOF（heredoc）

    • PHP 定界符 EOF 的作用就是按照原样，包括换行格式什么的，输出在其内部的东西；
    • 可以解析变量名，双引号内的转义字符，html格式，不能解析函数。

  • 类型比较

  • 字符串

    • strlen() - 字符串长度
    • strpos(“指定文本”,”指定查找内容”) - 返回文本中第一个指定内容的位置
    • 字符串链接，并置运算符

  • 条件判断

    • if
    • if…else…
    • if…elseif…else
    • switch

  • 数组

  • 排序

  • 超全局变量

    • $_REQUEST - 收集html表单提交的数据
    • $_POST
    • $_GET - 使用get时，变量会显示在url中

  • 魔术常量

    • _LINE_ - 当前行号
    • _FILE_ - 文件路径+文件名
    • _DIR_ - 文件目录
    • _FUNCTION_ - 函数名
    • _CLASS_ - 类名
    • _TRAIT_ - ？
    • _METHOD_ - 类的方法名
    • _NAMESPACE_ - 当前命名空间
